Тренажёр для подсчёта ожиданий и проверки выигрышных рук в риичи-маджонге

Инструкция:

1. Распаковать архив
2. Открыть файл index.html в браузере
3. Должно работать. Если не работает - можно понажимать F5 (не знаю, там секурити сесурити и пр.)

Картинки с тайлами взяты из репозитария https://github.com/fluffystuff/riichi-mahjong-tiles
